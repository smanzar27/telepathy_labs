# telepathy_labs

Prerequisite:
1. Install JDK 1.8
2. Ant 10.1 3 and set path directory 


Git Checkout

1. git clone https://github.com/smanzar27/telepathy_labs.git
2. git checkout master
3. git pull or fetch

How to run via ANT

1. Build: ant
2. Run ANT: 
i) ant QuestionOne
ii) snt QuestionTwo


How to run via JAR

1. Build: ant
2. Run JAR:
   i) java -jar QuestionOne Example1.txt
   ii) java -jar QuestionTwo ExampleOne.txt email,voice
